# cinema-web-record-collection
本项目用于收集和整理影视类网站的公开基础记录信息，对电影、电视剧及综合影视站点进行统一归档。仓库内容仅限站点信息层面的记录，不包含任何影视资源或播放入口。所有整理行为基于公开资料完成，项目定位为中性的信息整理工具，适合长期查询与对照使用。
